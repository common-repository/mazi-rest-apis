== Description ==
The plugin provides REST APIs getway with your WordPress site by sending and receiving data as JSON. It provides data access to the content of your site, and implements the same authentication restrictions

If you had website with Wordpress and need implements for your mobile app. It's for you.

Main features:
- List lastest post, support filtering/ordering
- List category
- View detail post
- Like a post
- Wishlist
- Basic settings

== Installation ==

= Manual Installation =

 1. Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation

 2. Activate 'Mazi REST APIs' through the 'Plugins' menu in WordPress.

= Better Installation =

 1. Go to Plugins > Add New in your WordPress admin and search for 'Mazi REST APIs'.

 2. Click Install.
